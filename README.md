## How to Run Locally (XAMPP)
1. Install XAMPP.
2. Copy the project folder into: `C:\xampp\htdocs\YOURFOLDER`
3. Start **Apache** in XAMPP Control Panel.
4. Open: `http://localhost/YOURFOLDER/`
